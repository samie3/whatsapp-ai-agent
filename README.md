# WhatsApp AI Agent — E-commerce SA

Powered by **Evolution API** + **Claude AI** + **n8n** + **Google Sheets**

## Stack
- Evolution API — WhatsApp QR connection
- Claude Sonnet — AI brain (Arabic + English)  
- n8n — automation workflows
- Google Sheets — lead tracker

## Flow
Customer WhatsApp → Evolution API → n8n → Claude AI → Reply + Log to Sheets

## Setup
1. Deploy to Railway
2. Set env vars (see .env.example)
3. Scan QR at /manager
4. Activate n8n workflow

Built for Saudi e-commerce. Bilingual Arabic + English.
